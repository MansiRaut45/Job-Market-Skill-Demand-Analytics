# Job Market Skill Demand Analytics

## Project Overview
This project analyzes job market data to identify the most in-demand skills, job trends, and hiring patterns using Python and Power BI.

## Objectives
- Identify most in-demand technical skills
- Analyze job trends and hiring patterns
- Visualize job market insights
- Build Power BI dashboard

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- Excel
- Power BI
- Jupyter Notebook
- Git & GitHub

## Dataset
The dataset contains job postings, skills, and job details collected from job portals.

(Dataset not uploaded due to large size)

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Skill Extraction
4. Data Analysis
5. Visualization
6. Dashboard Creation

## Key Insights
- Python and SQL are most in-demand skills
- Data Analyst and Data Scientist roles are highest demand
- Experience level affects salary significantly
- Job demand varies by location
