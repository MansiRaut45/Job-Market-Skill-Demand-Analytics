# Job Market Skill Demand Analytics

## Project Overview
This project analyzes job market data to identify the most in-demand skills, job trends, and hiring patterns using Python and Power BI.

## Objectives
- Analyze job postings dataset
- Identify most in-demand skills
- Analyze job trends
- Visualize insights using Power BI dashboard

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Excel
- Power BI
- Jupyter Notebook
- Git & GitHub

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Skill Extraction
4. Data Analysis
5. Data Visualization
6. Dashboard Creation

## Key Insights
- Identified top in-demand skills
- Analyzed job market trends
- Created interactive Power BI dashboard
